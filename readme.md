# 概要
M5StickC + Co2 HATを使用してCO2濃度の表示をしているのですが、Mac上でも数値を確認したいと思い作りました。  
ambient dataに送っているデータを取得し、メニューバーに表示しています。  
閾値を超えたらブリンクし、Macの通知でお知らせします。

# 使い方
ひとまずXcodeを使える方限定になります。  
apiUrlのXXXXXの部分にAmbientのチャネルIDとリードキーを設定してください。
